# Shrink Frontend

Shrink is a URL shortener application that allows users to shorten long URLs and generate custom short URLs. It also provides analytics on the number of clicks on each short URL.

## Features

- Shorten long URLs
- Generate custom short URLs
- Track click analytics
- Responsive design
- Dark mode

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Shadcn UI

## Installation

1. Clone the repository
2. Install dependencies

```bash
npm install
```

## Usage

```bash
npm run dev
```
